# vn-topic-planner (GitHub Pages)

Checklist theo **chủ đề** (mở rộng/thu gọn) + tick + ghi chú + thêm item nhanh (tự lưu trên trình duyệt).

## Chạy thử
- Mở `index.html` bằng trình duyệt.
- Hoặc VS Code → Live Server.

## Up lên GitHub Pages
1) Tạo repo mới  
2) Upload: `index.html`, `style.css`, `script.js`, `data.js`, `README.md`  
3) Settings → Pages → Deploy from branch → `main` / root.

## Chỉnh danh sách gốc
Sửa file `data.js`:
- `sections`: mỗi chủ đề có `id`, `name`, `emoji`, `hint`, `items`.

## Backup / chuyển máy
- Bấm **Xuất tiến độ (JSON)** để tải file.
- Máy khác bấm **Nhập tiến độ** để khôi phục.
